# NeoVim Config

This is my custom NeoVim config, designed to resemble the features and functionality of VSCode.

## Installation

To use this NeoVim config, follow these steps:

1. Clone this repository:

   `git clone https://github.com/PyDev19/neovim-config.git ~/.config/nvim`

2. If you want to preserve your current NeoVim config, make sure to back it up before proceeding.
